## 软件介绍
“我想要一个没广告的轻量级的学习五十音的App。”
而这便是我开发这个App的初衷。

## 主要功能
1. 日语五十音标准发音和书写
2. 卡片辅助记忆
3. 在线翻译
4. Pixiv插图欣赏

## 其他
没有广告
